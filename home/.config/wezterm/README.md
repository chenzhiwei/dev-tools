# Wezterm

## Shortcuts

* CTRL+SHIFT + F: CaseSensitive Search
* CTRL+SHIFT+ALT + ": 上下分屏
* CTRL+SHIFT+ALT + Up Arrow: 缩小上屏
* CTRL+SHIFT + Up Arrow: 激活上屏
* CTRL+SHIFT + Down Arrow: 激活下屏
* CTRL+SHIFT+ALT + %: 左右分屏
* CTRL+SHIFT+ALT + Left Arrow: 缩小左屏
* CTRL+SHIFT + Left Arrow: 激活左屏
* CTRL+SHIFT + right Arrow: 激活右屏
