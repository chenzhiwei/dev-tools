# README

## Pacman Use

```
# Install Package
pacman -S git

# Remove Package
pacman -Rs git

# Show Package
pacman -Qi git

# List Installed Packages
pacman -Qs

# Query File
pacman -Qo /etc/default/useradd

# List Package Files
pacman -Ql git

# Search Package
pacman -Ss git

# apt update
pacman -Sy

# Update System
pacman -Syu
```

## Disable Baloo

```
balooctl disable
balooctl purge
```
